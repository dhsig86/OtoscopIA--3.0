Passo-a-Passo
  1. Clique no botão "Escolha uma imagem" e selecione a imagem que deseja analisar.
  2. Use a ferramenta de recorte para selecionar a área da imagem que deseja classificar.
  3. Clique no botão "Cortar Imagem" para confirmar o recorte.
  4. Após o recorte, clique no botão "Classificar Imagem" para obter o resultado.
  5. O resultado será exibido com as classificações e probabilidades.
  6. Você pode clicar em "Imprimir Resultado" para imprimir a parte central do resultado.
  7. Clique em "Reset" para iniciar um novo processo.
